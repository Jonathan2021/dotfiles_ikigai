## Setup

- Application Launcher: `rofi`
- Music Player: `cmus`
- Shell: `bash`
- Terminal Emulator: `urxvt`
- Text Editor: `vim`
- Web Browser: `qutebrowser`
- Window Manager: `i3`


## Installation

If your're installing from scratch then just follow the [main documentation]().
